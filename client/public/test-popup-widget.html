üîß Troubleshooting

‚ùå Widget Not Appearing?

  • Open browser console (F12) and check for errors
  • Make sure you updated YOUR_CHATBOT_ID in the code below
  • Check that widget.js file exists at http://localhost:3000/widget.js
  • Verify your React dev server is running on port 3000

‚ùå "Failed to fetch" Error?

  • Make sure your FastAPI backend is running on https://vyomai.techraq.com
  • Enable CORS in your FastAPI backend (see code below)
  • Test if backend is reachable: curl https://vyomai.techraq.com
  • Check your firewall settings

üî• Backend CORS Fix (Required!)

Add this to your FastAPI backend:

from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],  # Or ["http://localhost:3000"]
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
