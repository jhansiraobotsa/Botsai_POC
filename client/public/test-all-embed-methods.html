Method 1: Popup Widget RECOMMENDED

A floating button appears in the corner. Click to open/close the chat. Perfect for all websites - non-intrusive and always accessible.

  • Floating button in corner
  • Click to open chat
  • Click minimize ‚Üí Shows button
  • Click button ‚Üí Reopens chat
  • Most user-friendly!

Embed Code:

<!-- Vyoma AI Popup Widget --> <script src="http://localhost:3000/widget.js" data-chatbot-id="YOUR_CHATBOT_ID" data-api-url="http://192.168.1.31:8006" data-position="bottom-right" data-color="#6366f1" async ></script>

Method 2: Iframe Embed ALWAYS VISIBLE

The chat is always visible in a fixed position. Users can minimize it to a button. Good for help centers or documentation pages.

  • Chat always visible
  • Fixed position on page
  • Click minimize ‚Üí Shows button
  • Click button ‚Üí Reopens chat
  • No initial button needed

Embed Code:

<!-- Vyoma AI Iframe Embed --> <iframe src="http://localhost:3000/widget.html?id=YOUR_CHATBOT_ID&api=http://192.168.1.31:8006" width="400" height="600" frameborder="0" style="position: fixed; bottom: 20px; right: 20px; border-radius: 16px; box-shadow: 0 10px 40px rgba(0,0,0,0.3); z-index: 9999;" ></iframe>

Method 3: Custom Script ADVANCED

Load the widget dynamically via JavaScript. Same as iframe but programmatically injected. Good for dynamic loading or A/B testing.

  • Programmatic injection
  • Dynamic loading control
  • Click minimize ‚Üí Shows button
  • Click button ‚Üí Reopens chat
  • Good for advanced uses

Embed Code:

<!-- Vyoma AI Custom Script --> <script> (function() { var iframe = document.createElement('iframe'); iframe.src = 'http://localhost:3000/widget.html?id=YOUR_CHATBOT_ID&api=http://192.168.1.31:8006'; iframe.style.cssText = 'position: fixed; bottom: 20px; right: 20px; width: 400px; height: 600px; border: none; border-radius: 16px; box-shadow: 0 10px 40px rgba(0,0,0,0.3); z-index: 9999;'; document.body.appendChild(iframe); })(); </script>
